Primary Source Verification

Moving PSV upstream to save hundreds of millions across the industry

Final Solutions / Primary Source Verification

The Problem

Each health insurance company individually has to verify their providers' data's legitimacy from many different primary sources, creating massive duplication of effort across the industry.

Current State Inefficiencies

These primary sources include state licensures, education records, malpractice databases, and many more. Providers tend to belong to many health plans, often around 16 to 20.

The Opportunity

What if we moved the PSV process upstream to be a part of CAQH's core data offering, so that health plans received already verified data?

CAQH is in a great position to solve this, being a non-profit that was founded to standardize provider data and already serves as a trusted intermediary for the industry.

Our Solution

We propose moving Primary Source Verification upstream to CAQH's platform, eliminating the need for individual health plans to conduct redundant verification processes.

Centralized Verification Model

Instead of 1,000+ organizations each verifying the same provider data independently, CAQH becomes the single point of verification for the entire industry.

Implementation Approach

Phase 1: Structured Data Integrations

Start with structured data integrations, like for malpractice records and state licensing databases

  • Direct API connections to state licensing boards
  • Integration with malpractice databases
  • Automated DEA verification systems

Phase 2: Unstructured Data Automations

Bring in unstructured data automations, like using OCR to verify medical license information

  • OCR for license verification
  • Document processing automation
  • AI-powered data extraction

Phase 3: Industry Partnerships

Establish partnerships with key data sources and verification agencies

  • Medical education institutions
  • Specialty board certifications
  • Hospital credentialing systems

Phase 4: Industry Standard

Establish this as a norm to unlock $100s of millions or more of savings across the healthcare insurance industry

  • Industry-wide adoption
  • Regulatory alignment
  • Continuous improvement processes
